# Resume Screening System

A Flask + Machine Learning project that compares resumes with job descriptions.

## Features

- Upload Resume PDF
- Upload Job Description
- PDF Text Extraction
- Text Cleaning
- TF-IDF Vectorization
- Cosine Similarity
- Resume Score
- Matching Skills
- Missing Skills

## Tech Stack

- Python
- Flask
- Scikit-Learn
- PyPDF2
- Bootstrap
- Pandas

## Installation

```bash
pip install -r requirements.txt
python app.py
```

## Project Workflow

Resume PDF
→ Text Extraction
→ Text Cleaning
→ TF-IDF
→ Cosine Similarity
→ Score Generation
